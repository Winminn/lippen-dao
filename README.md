# 1、简介
本项目主要是基于ibatis 或者 mybatis的通用dao实现，方便用户快速创建对像到数据表之间的关联。
# 2、说明
本项目是在工作时，于前辈(Exversion)的项目中剥离ibatis版本，并修改并添加mybatis版本。      由于mybatis支持annotation，于是有添加无配置版本的mybatis通用dao支持。
整个项目不是一个完整项目，只是一个通用工具，还不够完善，希望大家能给予意义。
